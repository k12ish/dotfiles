## Requirements

Binaries generally assumed to be installed on target machine.

Install these through `cargo` or maybe AUR

- `alacritty`
- `bat`
- `du-dust`, aka `dust`
- `fd-find`, aka `fd`
- `git-delta`, aka `delta`
- `ripgrep_all`, aka `rga`
- `ripgrep`, aka `rg`
- `tealdeer`, aka `tldr`
- `tokei`
- `zoxide`, aka `z`

Misc:

- `nvim` along with the `AstroNvim` repository
- `xclip`
- `diffpdf`

Commands 
```
pip install dateparser
```
